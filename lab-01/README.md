# Лабораторная работа №1 - Турнирная таблица

## Описание задания (Вариант 4)

Разработка графического интерфейса турнирной таблицы с следующими возможностями:
- Симметричное заполнение результатов матчей
- Автоматическое определение суммы очков для каждой команды
- Определение места команд в турнире
- Контекстное меню по правой кнопке мыши
- Поле создается на основе двумерного массива TextField-ов

## Функциональность

### Основные возможности:
- ✅ Создание турнирной таблицы произвольного размера (от 3 до 20 команд)
- ✅ Симметричное заполнение: при вводе результата A vs B автоматически заполняется B vs A
- ✅ Автоматический подсчёт очков (3 за победу, 1 за ничью, 0 за поражение)
- ✅ Сортировка команд по количеству очков
- ✅ Контекстное меню для операций с ячейками

### Контекстное меню:
- Очистить ячейку
- Очистить всю строку
- Очистить всю таблицу
- Показать статистику команды

## Технологии

- **Java(TM) SE Runtime Environment (build 23.0.2+7-58)**
- **Java HotSpot(TM) 64-Bit Server VM (build 23.0.2+7-58, mixed mode, sharing)**
- **Swing** с темой **FlatLaf**
- **Gradle 8.10** для сборки

## Запуск

```bash
# Запуск через Gradle
./gradlew run

# Сборка JAR файла
./gradlew jar

# Запуск JAR файла
java -jar build/libs/lab-01-1.0-SNAPSHOT.jar
```
